Lecture1 - المحاضرة 1

Hey class, you can download our favourite text editor Visual Code Studio form here. Your assignment will be making a profile webpage using HTML and CSS. See an example here.

Resources

  • FreeCodeCamp.org: sign up at this website, and then finish the challenges related to HTML/CSS only. There are several subjects over there, but for now, we only need to learn HTML and CSS. Later we will use it to learn JavaScript as well.
  • w3schools.com - HTML and w3schools.com - CSS these are great resources for beginners to learn web design.
  • How Web Browsers Work: watch this video for at least two times so that you get a general understanding of how the browsers actually work. Do not be intimidated by the information, just get a general understanding and that's it.

Assignment

Your assignment will be making your webpage that you made last week responsive, and then make it host it online on Github. See this video! How to host a website with Github

Resources

  • Responsive Web Design Fundamentals by Google: In this course you'll learn the fundamentals of responsive web design with Google's Pete LePage! You'll create your own responsive web page that works well on any device - phone, tablet, desktop or anything in between.
  • CSS Layout - float and clear and CSS3 Flexible Box Use of flexbox and float ensures that elements behave predictably when the page layout must accommodate different screen sizes and different display devices.
